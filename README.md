# 🌌 Astra Learn: Классификация звезд

**Astra Learn** — это интеллектуальная система для автоматического определения спектрального класса звезд. Приложение использует методы машинного обучения и компьютерного зрения для предсказания класса звезды (Галактика, Квазар или Звезда) как по табличным астрономическим признакам, так и по фотометрическим изображениям.
